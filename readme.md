# SaveAll
Utilidad para Guardar texto e imagenes del portapapeles. (Mas utilidades en versiones nuevas)

## Descripcion
SaveAll es una utilidad programada en Visual Basic, esta es capas de capturar el texto copiado del portapaeles e introducirlo en el programa para luego guardarlo. Tambien tiene la posibilidad de guardar imagenes copiadas en el portapapeles ( ya sea Click derecho>copiar o IMP PANT).

## Utilizacion
La utilizacion es sencilla con F2 se captura el texto del portapapeles y con F3 las imagenes (La ayuda se puede ver con F4).
El programa al minimizarlo queda en la barra de tareas para poder acceder a el rapidamente haciendo Click derecho-->Abrir. Para cerrarlo puedes cerrar el programa desde el Formulario o desde el icono de la barra de tareas haciendo Click derecho-->Salir. Indica cuando se guarda las imagenes y el texto mediante un globo de informacion. Dentro del Formulario del programa en la pestaña texto tenemos el texto copiado y un boton para guardar el texto en un archivo .txt en la ruta que necesites, en la pestaña Imagenes esta la Vista Previa de la ultima imagen guardada, tambien esta la seccion de acciones donde se puede limpiar el contenido del portapapeles y por ultimo en la seccion Imagenes Guardadas esta la lista de imagenes dentro de la carpeta "/APPPATH/Imagenes", en esta lista puedes seleccionar que foto abrir para que te aparesca el dialogo para elegir con que programa quieres abrir la imagen.
